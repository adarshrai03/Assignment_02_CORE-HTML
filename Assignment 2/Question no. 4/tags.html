HTML Tags

  1. DOCTYPE : html tag helps to Declares the document type and version of HTML being used

  2. head : Contains meta-information about the document, like title, character encoding, links to stylesheets, and more.

  3. title : Sets the title of the web page (displayed in the browser's title bar or tab).

  4. meta : Provides metadata about the HTML document, such as character encoding, author, and viewport settings.

  5. link : Links external resources like stylesheets to the HTML document.

  6. nav : Contains navigation links.

  7. section : Defines a section of a document, such as a chapter, a tab panel, or any other grouping of content.

  8. p : Represents a paragraph.

  9. a : Creates hyperlinks to other web pages or resources.

  10. img : Embeds images in the document.
